# Insta-DM-Bot

A simple bot to send a Message to Multiple User Accounts.

# Dependencies

1. pip install eel.

2. pip install selenium.

3. Google Chrome (Version 84.0.4147.125 (Official Build) (64-bit)).

# Usage 

1) Clone The Repo
2) Install the dependencies by using Pip
3) Run the bot.py
